To run the program:

$python AffineGapALign.py inputfilename

To save the output in a file:

$python AffineGapALign.py inputfilename > outputfilename

To run your own sequences, enter them in the file "Sequence.txt".

Sample:

>seq0
FQTWEEFTSRA     //the first sequence
>seq1
KYRTWEEAAFTRA   //the second sequence
